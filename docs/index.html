What is metacog?这是什么？

metacog is a single Claude Code skill that packs 8 metacognitive capabilities into one /metacog command. Instead of just answering your questions, it helps you think better about thinking itself.

大多数 AI 交互是：用户提问 → AI 回答。但真正的瓶颈往往不是"答案是什么"，而是"我问对问题了吗？" metacog 反转了这个模式：分析思维过程本身，帮用户升级认知能力。

Gottman's Meta-Emotion Philosophy (1996)
Your attitude toward emotions determines regulation | 你对情绪的态度决定调节效果

Recognize → Name → Understand

Design Philosophy设计哲学

01
Single File Architecture | 单文件架构
All 8 skills + controller in one SKILL.md. Users remember one command: /metacog. The router handles complexity; users describe their situation naturally.
02
Bilingual Headers | 双语标题
All output sections use ### English | 中文. English ensures LLM processing precision; Chinese provides intuitive understanding. Works for both audiences.
03
Confidence Scoring | 置信度评分
Every output includes 1-10 score with justification. Directly combats Dunning-Kruger effect — forces calibration of certainty for both AI and user.
04
Anti-Flattery Principle | 反媚原则
Every output must survive: "If I delete all compliments and chicken soup, what's left?" If nothing remains, the output failed. Substance is respect.
05
Reusable Principles | 可复用原则
Every interaction extracts one transferable principle. The goal isn't solving one problem — it's giving you a tool that works across many problems.
06
Academic Grounding | 学术根基
Not built on vibes. Every skill maps to peer-reviewed frameworks: Flavell, Kahneman, Gottman, Bateson, Argyris, Shinn. Theory drives design.

Anti-Patterns反模式

These outputs are explicitly forbidden across all skills. 所有技能严禁以下输出模式：

Flattery | 恭维
✗ "你能思考这个问题说明你很有自省力"
✓ Jump straight to analysis
Chicken Soup | 鸡汤
✗ "相信自己，你一定可以的"
✓ Give concrete executable steps
Hedging | 敷衍
✗ "这要看具体情况"
✓ List key variables + conditional logic
Emotion Denial | 否认情绪
✗ "别想太多"
✓ Name the emotion precisely, trace effects
Naive Advice | 天真建议
✗ "说实话就好了"
✓ Analyze dynamics, give specific phrasing
Flattening | 扁平化
✗ "都很重要"
✓ Explicit priority ranking
Hallucination | 幻觉填充
✗ Making up answers when info is missing
✓ State what's missing, then ask
